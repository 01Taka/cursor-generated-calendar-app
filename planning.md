# UserProfile
- カレンダーを開くたびに「今日も埋まっている」と感じ、自分の時間が他人の予定で支配されているように感じる。
- 予定のタイトルだけでは、その時間を何のために使うのかが思い出せず、終わったあとに虚無感を感じやすい。
- 深い集中が必要な仕事をしているが、会議と雑務の隙間にしか作業時間が取れない。
- 週の終わりに「何を達成したか」より「何に時間を使ったか」が分からず、振り返りに不安を感じる。
- デジタルツールに慣れており、ドラッグ操作やキーボードショートカットによる素早い入力を好む。
- 他人との予定調整より、自分一人の時間設計が主な用途である。

# EmotionalValue
- カレンダー全体への漠然とした不安が減り、今日の時間の「意味」に対する安心感が増える。
- 予定が多いことへの圧迫感が減り、自分が選んだ意図に沿っているという手応えが増える。
- 週末の振り返りでの後悔感が減り、時間の使い方に対する納得感が増える。
- 空白の時間への罪悪感が減り、回復や深い仕事のための余白への正当感が増える。

# CoreExperience
- 一日の時間軸上に、各ブロックが「意図（インテント）」の色とラベルで並んで見える。
- ブロックをドラッグして開始・終了時刻を変え、意図カテゴリを付け替えた結果が即座に週全体の配分に反映される。
- 週の冒頭に、各意図タイプへ割き当てたい時間の目標を設定し、実績との差分が数値と色で示される。
- 一日の終わりに、その日に記録したブロックを一覧で見返し、翌日へ持ち越す未完了の意図を選んで次の日に配置する。
